# 【2026年版】日本の障害物レース（OCR）完全ガイド — スパルタンレース・HYROXの始め方

「走るだけのマラソンじゃ物足りない」「ジムのトレーニングを実戦で試したい」。そんな人に今、爆発的に広がっているのが**障害物レース（OCR: Obstacle Course Race）**です。

壁を乗り越え、泥をくぐり、重いものを運ぶ。全身をフルに使うレースは、完走した瞬間の達成感が桁違い。2026年の日本では、世界的ブランドのスパルタンレースとHYROXが各地で開催され、初心者でも気軽に挑戦できる環境が整っています。

この記事では、日本国内で参加できるOCRの全体像から、準備、当日の流れ、さらにレース前後の旅行プランまで、まるごとお伝えします。

---

## OCR（障害物レース）とは？

OCRとは「Obstacle Course Race」の略で、ランニングコース上に設置されたさまざまな障害物をクリアしながらゴールを目指す競技です。

壁の登攀、ロープクライミング、重量物の運搬、泥水くぐりなど、種目は多岐にわたります。単純な走力だけでなく、筋力・持久力・メンタルの強さが問われるのが特徴です。

「アスリートじゃないと無理では？」と思うかもしれませんが、実際には運動初心者や親子連れの参加者も多数。障害物をクリアできなくても代替種目（バーピーなど）で先に進めるルールがあるため、完走率は非常に高いのが実情です。

### Spartan Race Japan（スパルタンレース）

2007年にアメリカで設立され、2010年に第1回レースを開催。現在は世界40か国以上・年間170レース以上を展開する世界最大級のOCRです。日本では2017年に初上陸し、**2026年は上陸10年目のシーズン**。

### HYROX（ハイロックス）

ドイツ発の屋内フィットネスレースで、ランニングとファンクショナルトレーニングを融合した新しい形の耐久競技。**2025年8月に横浜で日本初上陸**を果たし、約3,800人が参加する大盛況となりました。

#### 競技形式

1kmのランニングと1種目のワークアウトを**交互に8セット**繰り返す、合計約8kmのレースです。

**8つのワークアウト種目**:

1. スキーエルゴ（1,000m）
2. スレッドプッシュ（50m）
3. スレッドプル（50m）
4. バーピーブロードジャンプ（80m）
5. ローイング（1,000m）
6. ファーマーズキャリー（200m）
7. サンドバッグランジ（100m）
8. ウォールボール（100回）

#### 参加カテゴリ

| カテゴリ | 内容 |
|---------|------|
| **Single** | 1人で全種目を完走 |
| **Doubles** | 2人チーム。ランは一緒に、ワークアウトは分担 |
| **Relay** | 4人チーム。1人あたりラン1km＋ワークアウト2種目 |

重量設定は「Open（一般）」と「Pro（上級）」の2段階があり、初心者はOpenでの参加が推奨されます。

## 初心者はどれから始めるべき？

レース選びに迷ったら、以下を参考にしてください。

**アウトドアで泥まみれになりたい！**
→ **スパルタンレース Sprint**（5km / 障害物20個）がおすすめ。自然の中を走り、壁や綱を使った障害物に挑む野外レースの醍醐味を味わえます。

**屋内で天候を気にせず挑戦したい！**
→ **HYROX（Relay）** がおすすめ。HYROXは全カテゴリを通じて完走率99%以上を誇るレースです。中でも4人チームのRelayなら1人あたりの負担が軽く、仲間と一緒に盛り上がれます。

**とにかくソロで限界に挑みたい！**
→ **HYROX Single** または **スパルタンレース Super** へ。ある程度の走力と筋力があれば十分に完走できます。

**子どもと一緒に参加したい！**
→ **スパルタンレース Kids**。1〜3kmのコースで、親子で楽しめます。川崎（4月）や千葉（5月）で開催予定です。

## 必要な準備・装備

### トレーニング（レース2〜3か月前から）

- **ランニング**: 週2〜3回、5〜10kmのジョギング
- **筋力トレーニング**: 懸垂・バーピー・スクワット・デッドリフト
- **グリップ力**: ぶら下がりトレーニング（スパルタンレース向け）

### 当日の服装・持ち物

| アイテム | ポイント |
|---------|---------|
| **シューズ** | トレイルランニングシューズ（スパルタン）/ 室内用ランシューズ（HYROX） |
| **ウェア** | 速乾性のあるタイツ＋Tシャツ。スパルタンは汚れてもよいもの |
| **グローブ** | スパルタンでは軍手やトレーニンググローブが便利 |
| **着替え** | レース後用の衣類・タオル・ビニール袋（泥だらけの服を入れる） |
| **日焼け止め** | 屋外レースでは必須 |
| **補給食** | Beast以上の長距離はエナジージェル推奨 |

## 大会当日の流れ

### スパルタンレースの場合

1. **受付（スタート1〜2時間前）**: IDを提示し、ゼッケン・計測チップ・ヘッドバンドを受け取る
2. **荷物預け**: 貴重品以外をまとめて預ける
3. **ウォームアップ**: スタートエリア近くで軽く体を動かす
4. **スタート**: ウェーブ制（時間帯ごとのグループ出走）
5. **レース**: 障害物をクリアしながらコースを走破。失敗したらバーピー30回
6. **ゴール**: 最後のファイヤージャンプを飛び越えてフィニッシュ。メダル授与
7. **フェスティバルエリア**: フード・ドリンク・物販を楽しむ

### HYROXの場合

1. **受付（スタート1時間前）**: オンラインチェックイン後、会場で計測チップを受け取る
2. **ウォームアップエリア**: 屋内に専用スペースあり
3. **スタート**: ウェーブ制で出走
4. **レース**: 1kmラン→ワークアウト→1kmラン→ワークアウト...を8セット
5. **ゴール**: タイムが記録され、世界ランキングに反映
6. **アフターパーティー**: DJ・フード・ドリンクで参加者同士の交流

## レース前後の宿泊・観光プラン

せっかく遠征するなら、レースだけで帰るのはもったいない。前後に宿泊して観光を楽しむのがおすすめです。

### 千葉（東京ドイツ村 / 幕張メッセ）

- **前泊**: 木更津や海浜幕張のホテルに泊まり、前日はゆっくり準備
- **レース後**: マザー牧場や鴨川シーワールドで千葉を満喫。アウトレットでショッピングも
- **温泉**: 木更津温泉や竜宮城スパで疲れた体を癒やす

### 新潟・妙高（LOTTE ARAI RESORT）

- **前泊**: ロッテアライリゾート自体がホテル併設。前日入りして施設を楽しめる
- **レース後**: 妙高高原の温泉郷（赤倉温泉、燕温泉）でリカバリー。秋は紅葉も絶景
- **グルメ**: 新潟の日本酒と新鮮な魚介を堪能

### 静岡・裾野（スノーパークイエティ）

- **前泊**: 御殿場エリアのホテルがアクセス便利
- **レース後**: 御殿場プレミアム・アウトレットや富士サファリパークへ。天気がよければ富士山の絶景が間近に
- **温泉**: 御殿場エリアには日帰り温泉施設が豊富

### 大阪（インテックス大阪）

- **前泊**: なんばや梅田のホテルに泊まり、前夜は大阪グルメを堪能
- **レース後**: 道頓堀、大阪城、USJなど観光スポットが充実
- **グルメ**: たこ焼き、お好み焼き、串カツ。カーボローディングにも最適
