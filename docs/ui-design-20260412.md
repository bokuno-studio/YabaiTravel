# yabai.travel 次期版 UIコンポーネント設計書 (2026-04-12)

本ドキュメントは、次期版 `yabai.travel`（一覧ページのみの構成）における主要UIコンポーネントの設計仕様である。

## 1. デザインガイドライン
- **ベースカラー**: `oklch(0.62 0.14 30)` (暖色系レッド/オレンジ) をプライマリに使用。
- **カード形状**: 角丸 `var(--radius)`、上部にレースタイプ別のアクセントボーダー。
- **タイポグラフィ**: `font-sans`、階層を明確にするためのウェイト調整。
- **インタラクション**: ホバー時のシャドウ強調、アコーディオンによる情報の段階的開示。

## 2. EventCard コンポーネント

全情報を集約した高密度カード。詳細ページへの遷移は行わず、このカード内で全ての情報を完結させる。

### Props 定義
```typescript
interface EventCardProps {
  event: EventWithCategories;
  lang: 'ja' | 'en';
  t: (key: string) => string;
}
```

### 実装コード案
```tsx
import React, { useState } from 'react';
import { Calendar, MapPin, Ruler, ExternalLink, Map as MapIcon, ChevronDown, ChevronUp } from 'lucide-react';
import { Card, CardContent, CardFooter } from '@/components/ui/card';
import { Badge } from '@/components/ui/badge';
import { Button } from '@/components/ui/button';
import { cn } from '@/lib/utils';
import type { EventWithCategories } from '@/types/event';

export const EventCard: React.FC<EventCardProps> = ({ event, lang, t }) => {
  const [isExpanded, setIsExpanded] = useState(false);
  const isEn = lang === 'en';

  // 言語別フィールドの解決
  const name = isEn ? (event.name_en ?? event.name) : event.name;
  const country = isEn ? (event.country_en ?? event.country) : event.country;
  const location = isEn ? (event.location_en ?? event.location) : event.location;
  const description = isEn ? (event.description_en ?? event.description) : event.description;

  // 申込ステータスの判定
  const getEntryStatus = () => {
    const now = new Date();
    const start = event.entry_start ? new Date(event.entry_start) : null;
    const end = event.entry_end ? new Date(event.entry_end) : null;

    if (start && now < start) return 'not_yet';
    if (end && now > end) return 'closed';
    if (start && end && now >= start && now <= end) return 'open';
    return 'unknown';
  };

  const status = getEntryStatus();
  const statusConfig = {
    open: { label: t('status.open'), color: 'bg-green-500 hover:bg-green-600', button: t('button.entry_now'), active: true },
    not_yet: { label: t('status.not_yet'), color: 'bg-amber-500 hover:bg-amber-600', button: t('button.entry_not_yet'), active: false },
    closed: { label: t('status.closed'), color: 'bg-red-500 hover:bg-red-600', button: t('button.entry_closed'), active: false },
    unknown: { label: t('status.unknown'), color: 'bg-gray-400 hover:bg-gray-500', button: t('button.entry_page'), active: !!event.entry_url },
  };

  const currentStatus = statusConfig[status];

  // 距離情報の整形
  const distances = event.categories
    ?.map(c => c.distance_km)
    .filter((d): d is number => d !== null)
    .sort((a, b) => a - b)
    .map(d => `${d}km`)
    .join(', ');

  // 紹介文の制御
  const truncatedDesc = description?.slice(0, 100);
  const hasMore = (description?.length ?? 0) > 100;

  return (
    <Card className={cn(
      "group relative overflow-hidden flex flex-col h-full border-t-4 transition-all duration-300 hover:shadow-lg",
      // レースタイプ別のボーダーカラー（既存の raceTypeBorders を流用）
      getRaceTypeBorder(event.race_type)
    )}>
      <CardContent className="p-5 flex-1 space-y-4">
        {/* 1 & 2. ヘッダー（バッジ） */}
        <div className="flex justify-between items-start gap-2">
          <Badge variant="outline" className={cn("text-[10px] uppercase font-bold px-2 py-0.5", getRaceTypeBadgeStyle(event.race_type))}>
            {event.race_type}
          </Badge>
          <Badge className={cn("text-[10px] text-white border-0 px-2 py-0.5", currentStatus.color)}>
            {currentStatus.label}
          </Badge>
        </div>

        {/* 3. レース名 */}
        <h3 className="text-lg font-bold leading-tight line-clamp-2 min-h-[3rem]">
          {name}
        </h3>

        {/* 4, 5, 6. メタ情報 */}
        <div className="space-y-2 text-sm text-muted-foreground">
          {/* 日程 */}
          <div className="flex items-center gap-2">
            <Calendar className="w-4 h-4 shrink-0 text-primary" />
            <span>{formatDateRange(event.event_date, event.event_date_end, lang)}</span>
          </div>
          {/* 場所 */}
          <div className="flex items-center gap-2">
            <MapPin className="w-4 h-4 shrink-0 text-primary" />
            <span className="truncate">{country} / {location}</span>
          </div>
          {/* 距離 */}
          <div className="flex items-center gap-2">
            <Ruler className="w-4 h-4 shrink-0 text-primary" />
            <span className="font-medium text-foreground">{distances || '---'}</span>
          </div>
        </div>

        {/* 7. 紹介文（アコーディオン） */}
        <div className="relative pt-2 border-t border-border/50">
          <p className={cn("text-sm leading-relaxed text-foreground/80", !isExpanded && "line-clamp-3")}>
            {description}
          </p>
          {hasMore && (
            <button 
              onClick={() => setIsExpanded(!isExpanded)}
              className="mt-2 flex items-center gap-1 text-xs font-bold text-primary hover:underline transition-all"
            >
              {isExpanded ? (
                <><ChevronUp className="w-3 h-3" /> {t('common.show_less')}</>
              ) : (
                <><ChevronDown className="w-3 h-3" /> {t('common.show_more')}</>
              )}
            </button>
          )}
        </div>
      </CardContent>

      {/* 8, 9, 10. フッター（アクション） */}
      <CardFooter className="p-5 pt-0 flex flex-wrap items-center gap-3 border-t border-border/30 mt-auto">
        {/* 申込ボタン */}
        <Button 
          className={cn(
            "flex-1 font-bold min-w-[140px]", 
            currentStatus.active ? "bg-primary text-white" : "bg-muted text-muted-foreground pointer-events-none"
          )}
          onClick={() => event.entry_url && window.open(event.entry_url, '_blank')}
          disabled={!currentStatus.active}
        >
          {currentStatus.button}
        </Button>

        {/* 補助リンク */}
        <div className="flex items-center gap-4 ml-auto">
          {event.official_url && (
            <a href={event.official_url} target="_blank" rel="noopener" className="text-muted-foreground hover:text-primary transition-colors" title={t('common.official_site')}>
              <ExternalLink className="w-5 h-5" />
            </a>
          )}
          {event.latitude && event.longitude && (
            <a 
              href={`https://www.google.com/maps?q=${event.latitude},${event.longitude}`} 
              target="_blank" 
              rel="noopener" 
              className="text-muted-foreground hover:text-primary transition-colors"
              title={t('common.view_map')}
            >
              <MapIcon className="w-5 h-5" />
            </a>
          )}
        </div>
      </CardFooter>
    </Card>
  );
};
```

## 3. FilterBar コンポーネント

デスクトップでは上部またはサイド、モバイルではドロワーまたは折りたたみ式で表示。

### Props 定義
```typescript
interface FilterState {
  dateRange: [string | null, string | null];
  countries: string[];
  raceTypes: string[];
  distanceRange: string; // 'all' | 'short' | 'middle' | 'long'
  entryOpenOnly: boolean;
}

interface FilterBarProps {
  filters: FilterState;
  onChange: (filters: FilterState) => void;
  options: {
    countries: string[];
    raceTypes: string[];
  };
  t: (key: string) => string;
}
```

### 実装案
- **初期表示**: `entryOpenOnly: true` をデフォルトに設定。
- **カレンダー**: `Popover` + `Calendar` (mode="range")。
- **マルチセレクト**: `Checkboxes` または `Select` (Multiple)。
- **トグル**: `Switch` (Entry Status)。

## 4. 変更点まとめ（旧バージョンとの比較）

| 要素 | 旧バージョン | 次期版 |
|------|------------|--------|
| **カードサイズ** | コンパクト (詳細ページへ誘導) | フルサイズ (全情報完結型) |
| **距離表示** | ツールチップ / 最小距離のみ | 全カテゴリをテキスト列挙 |
| **紹介文** | 100文字程度で固定 | 600文字まで対応 (アコーディオン) |
| **ボタン** | 単一の「詳細」ボタン | 申込・公式サイト・マップの3系統 |
| **フィルター** | 多岐にわたる項目 | 厳選された5項目、デフォルト「申込中」 |

---

## 5. コーディング時の注意点
1. **多言語対応**: `name_en` 等の有無を必ず確認し、フォールバックを適切に行う。
2. **パフォーマンス**: フィルター変更時の再レンダリングを最適化（`useMemo` 等の活用）。
3. **アクセシビリティ**: ボタンの `disabled` 状態やスクリーンリーダー向けのラベル（`aria-label`）を付与。
