# 大会データ項目仕様

各大会・レースで管理する項目の一覧。  
**大会単位**と**カテゴリ単位**で保持する項目が異なる。詳細は [データ構造設計](./SPEC_DATA_STRUCTURE.md) を参照。

**レースの種類**: 「こんなレースも入れたい」が出るたびに不定期で追加していく。

---

## スコープの整理

| スコープ | 説明 | 例 |
|----------|------|-----|
| **大会（Event）** | 同一開催の全カテゴリで共通 | 日付、場所、アクセス、コースマップ |
| **カテゴリ（Category）** | カテゴリ毎に異なる | 獲得標高、制限時間、申込み費用 |

## 大会単位の項目

### 基本情報

| 項目 | 説明 |
|------|------|
| 大会名 | |

| 場所 | 開催地 |
| 申込みサイトURL | |

| 項目 | 説明 |
|------|------|
| 申込み開始 | |

### 東京からのアクセス（往路）

**主軸: 公共交通機関でどうやって行くか**

| 項目 | 説明 |
|------|------|
| 経路・乗り換え情報 | 公共交通機関のルート、乗り換えの詳細を表示 |
| トータル時間の目安 | 東京から会場までの所要時間 |
| 費用概算 | |
| 現金必須 | 公共交通機関で現金必須の区間があればその旨 |
| 予約サイトリンク | バス・船等でオンライン予約がある場合のリンク（飛行機等の無数にあるものは不要） |
| 前泊推奨地 | スタートに間に合う拠点 |
| 宿泊費用目安 | 星3レベルでの当該日の平均的な宿泊費用 |

### 公共交通で行けない場合

| 項目 | 説明 |
|------|------|
| シャトルバス | 有無 |
| タクシー | シャトルも無い場合の最寄り駅からの距離・費用概算 |

### 東京までのアクセス（復路）

| 項目 | 説明 |
|------|------|
| 経路・乗り換え情報 | 会場から東京までのルート、乗り換え |
| トータル時間の目安 | レース終了時間から東京帰着までの所要時間 |

### その他

| 項目 | 説明 |
|------|------|
| コースマップ | 画像またはPDFで保管 |
| 使用禁止品 | 大会共通で禁止されているもの |
| ふるさと納税申込リンク | 開催地でふるさと納税をやっている場合の申込URL |

## データ収集・格納方針

- バックグラウンドで収集
- **確定情報のみ格納**: 公式で発表されている情報のみ
- **更新は原則なし**。既存データの上書きはしない
- データが取れなかった項目は、**追加で探しに行って埋める**のはあり

### 取得元の方針

- **手動入力はなし**
- API が使える項目は API で取得
- 基本は Google（検索・Maps・Directions 等）が主なソースになる想定

詳細は [データ構造設計](./SPEC_DATA_STRUCTURE.md) の「収集・格納の原則」を参照。
