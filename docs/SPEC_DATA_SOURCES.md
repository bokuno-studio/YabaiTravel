## レース種別の追加方針

- マスタを直接編集するのではなく、**ユーザーが Cursor に依頼**する前提
- 会話例: 「なんのレース追加する？」「この URL から取得するつもりだけどどう思う？」
- その都度、取得元 URL や方式を相談しながら追加
