# yabai.travel サイト骨子案（仮）

## 1. コンセプト

**Yet Another Badass Adventure Itinerary（またしてもヤバい冒険の旅程）**

トレラン、スパルタンレース、ハイロックスなどの過酷なエンデュランス系大会へ挑む人へ向けた、「大会情報＋参戦ロジスティクス」の統合ポータルサイト。

## 2. 解決する課題（ペインポイント）

- **アクセスの不透明さ**: 登山口や地方のマイナーな開催地へ、「東京からどうやって行くのが最適か」を調べるのが面倒
- **宿泊のハードル**: 早朝スタートの大会において、「どこに泊まればスタート地点に間に合うか」が分からない
- **日程の判断**: 「日帰りできるのか」「前泊必須か」「ダメージを考慮して後泊すべきか」の判断基準がない

## 3. コア機能

1. **変態レースカレンダー**: トレラン、障害物レース、アドベンチャーレース等の開催日・エントリー期間の一覧
2. **東京発・最適アクセスガイド**: 車、新幹線＋レンタカー、夜行バスなど、東京を起点とした現実的な移動ルートと所要時間の提示
3. **ステイタス（宿泊）判定**: 各大会に対して「日帰り可能」「前泊必須」「後泊推奨」のフラグ付けと、拠点とすべき宿泊エリアのサジェスト

## 4. 技術スタック

- **フロント**: React + TypeScript + Vite（Vercel デプロイ）
- **バックエンド/DB**: Supabase
- **データ収集**: 別途実装（方式は任せる）
- **開発フロー**: チケット駆動、Mermaid 等でデータ構造・画面遷移を可視化
